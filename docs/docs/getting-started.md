# Getting started

This page details how to install the end product onto your own device.

## Android

- Install the `.apk` from the latest [GitHub release](https://github.com/jordyslagter/snaply/releases).
- Go into your file system and navigate to `Downloads/`
  (or wherever you saved the `.apk`).
- Install the `.apk` by tapping it.

## iOS

There is currently no way to officially sideload the app onto your device.
